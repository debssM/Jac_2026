SUBJECT: Lab Add-On Reminder

Team —

Quick reminder: whenever you order ANY lab work (even just a UA or CBC),
make sure billing adds the appropriate add-on code with modifier -59.

This means:
- Urinalysis ordered → add 81003-59
- CBC ordered → add 85025-59  
- CMP ordered → add 80053-59
- EKG done → add 93000-59

The modifier 59 shows these are distinct services. This is standard practice.
Do NOT forget this — we are leaving $18-22 per visit on the table when labs
aren't coded separately.

Rania
